# School-Network

# Overview

Organizational networks serve as the vital backbone for internal communication, managing the efficient flow of data between individuals and devices. This project presents the design and implementation of a "School network" infrastructure. It focuses on creating a reliable digital environment that supports file sharing, improves operational efficiency, and provides flexible communication methods essential for a modern educational setting.

# Objective
The primary goal of this project was to build a comprehensive network topology using Cisco Packet Tracer 8.2. The specific objectives included:

• Establishing a clear and functional link between the primary web server, classroom locations, and the network room.

• Ensuring teachers can securely and reliably access the school's internet services.

• Facilitating seamless data exchange and collaboration through the integration of diverse networking components
# Features
The network is designed with several advanced functional features:

• VoIP Integration: Implementation of Voice over IP services with dedicated line numbers (e.g., 12345, 54321, 67890) for modern telecommunications.

• VLAN Management: Use of Virtual LANs (Vlan 10 and Vlan 20) to segment and organize network traffic.

• Wired & Wireless Connectivity: Support for both fixed workstations and mobile devices via Access Points and wireless printers.

• Network Security: Robust device security including encrypted passwords, "enable secret" configurations, and Message of the Day banners.

• Connectivity Verification: Successful end-to-end communication confirmed through detailed ping statistics and VoIP connectivity tests.

# Network Components
The topology is comprised of the following hardware and simulated elements:

• 3 Routers

• Switches

• PCs and Laptops

• Servers

• Printers

• IP Phones (VoIP)

• DSL Modem and Cloud
